projeto PING-PONG (p5js)

Este projeto consiste na implementação do jogo Pong, utilizando a biblioteca p5.js, uma ferramenta poderosa baseada em JavaScript para o desenvolvimento de gráficos interativos. O objetivo do jogo é criar uma versão digital de pingue-pongue, onde dois jogadores controlam palhetas para tentar marcar pontos ao fazer a bola passar pela palheta do adversário.

A abordagem deste projeto foca na aplicação de conceitos de programação, como manipulação de objetos, interação com o usuário e implementação de mecânicas de colisão. Além disso, o jogo é projetado para ser simples e intuitivo, com uma interface gráfica limpa e controles diretos, proporcionando uma experiência agradável para o usuário.Este jogo é projetado para dois jogadores, cada um controlando uma palheta na tela e competindo para marcar pontos. O objetivo é evitar que a bola ultrapasse a sua palheta, enquanto tenta fazer com que a bola passe pela palheta do adversário.

Jogador 1 (Controle da Palheta Esquerda): O jogador 1 usa as teclas W e S para controlar sua palheta. A tecla W move a palheta para cima. A tecla S move a palheta para baixo. A palheta do jogador 1 se move verticalmente dentro dos limites da tela, e sua função é interceptar a bola para evitar que ela ultrapasse sua posição. Jogador 2 (Controle da Palheta Direita): O jogador 2 utiliza as setas direcionais do teclado para controlar sua palheta. Seta para cima: Move a palheta para cima. Seta para baixo: Move a palheta para baixo. Assim como o jogador 1, o objetivo do jogador 2 é bloquear a bola com sua palheta e tentar fazer com que a bola ultrapasse a palheta do adversário.

projeto filme (p5js)

Este projeto em p5.js é uma aplicação interativa simples e divertida que recomenda filmes com base na idade do usuário e em suas preferências por gêneros como fantasia e aventura. Ele utiliza uma interface gráfica com elementos HTML integrados ao canvas do p5.js, permitindo que o usuário insira sua idade e marque caixas de seleção para indicar seus gostos pessoais.A recomendação de filme aparece diretamente no centro da tela, com um estilo visual agradável, tornando a experiência intuitiva e envolvente. A ideia é simular uma espécie de "assistente de escolha de filmes" que ajuda o usuário a decidir o que assistir, especialmente voltado para um público jovem ou familiar.

Ao abrir o programa, a pessoa digita sua idade e marca se gosta de filmes de fantasia e/ou aventura. Depois, é só clicar no botão "Ver recomendação". O programa então lê essas informações e escolhe um filme de acordo com a idade e os gostos da pessoa. Por exemplo, se a pessoa for criança e gostar de fantasia, o programa pode sugerir Harry Potter. Se for adolescente, pode sugerir Extraordinário ou A Culpa é das Estrelas. A recomendação aparece no meio da tela, com letras grandes e coloridas. O objetivo é ajudar de forma divertida a escolher um filme legal para assistir.

projeto color follow (p5js)

Este projeto cria uma bolinha que segue o movimento do mouse na tela. Quando o programa é iniciado, a tela é criada com fundo verde e a bolinha começa no centro da tela. A cor da bolinha também é gerada de forma aleatória, com valores para as cores vermelho, verde e azul. À medida que o usuário move o mouse, a bolinha começa a se mover em direção ao cursor, se aproximando gradualmente de onde o mouse está. O movimento é feito de forma suave, com a bolinha se ajustando lentamente à posição do mouse tanto na direção horizontal (esquerda/direita) quanto vertical (cima/baixo).

Além disso, sempre que o botão do mouse é pressionado, a cor da bolinha muda para uma nova cor aleatória. A bolinha pode ficar com diferentes tonalidades, já que sua cor é determinada de forma aleatória para cada componente RGB (vermelho, verde e azul), e a transparência também muda aleatoriamente, tornando a bolinha mais ou menos visível. Isso cria um efeito visual interessante e dinâmico.
